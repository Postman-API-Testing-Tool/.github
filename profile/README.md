## 🚀 Getting Started with API Testing

*Prerequisites: A Windows PC with internet connection and API endpoints to test.*

1.  **Install Postman:** Complete your postman download and install this api testing tool on your Windows system.
2.  **Create Your First Request:** Launch this rest api tester, enter a URL, and click Send to see the API response.
3.  **Save to Collection:** Click Save in this http client to add your request to a collection for future use.
4.  **Write a Test:** Navigate to the Tests tab in this api development tool and add a script like `pm.test("Status is 200", () => pm.response.to.have.status(200));`
